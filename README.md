# Capstone-Project


  **Foodie Inc. - Food Delivery Platform**

Comprehensive Project Documentation

Project Overview

Foodie Inc. is a full-stack food delivery platform that connects customers with local restaurants. Built with modern web technologies, it provides a seamless experience for browsing restaurants, viewing menus, and placing orders online.

Current Status

- Backend API: Fully operational Spring Boot application
- Frontend: Angular customer interface running
- Database: H2 in-memory database with sample data
- Security: Basic authentication and CORS configuration
- Documentation: Swagger API documentation available

Architecture Overview

Tech Stack

Backend (Spring Boot 3.1.5)

- Framework: Spring Boot with Java 17
- Database: H2 (development) / MySQL (production)
- Security: Spring Security with JWT tokens
- API Documentation: Swagger/OpenAPI 3
- Build Tool: Maven 3.8+
- ORM: Spring Data JPA with Hibernate

Frontend (Angular 20.3)

- Framework: Angular with TypeScript
- Styling: Custom CSS with responsive design
- HTTP Client: Angular HttpClient for API communication
- Routing: Angular Router for navigation
- Build Tool: Angular CLI

Database Schema

- Primary: MySQL 8.0 (production)
- Development: H2 in-memory database
- ORM: JPA/Hibernate with automatic schema generation
